# OBSCURA

A brutalist streetwear website concept created by NexBoom Studios.  
NexBoom Studios tarafından geliştirilen brutalist streetwear website konsepti.

---

## Overview / Proje Hakkında

OBSCURA is a premium streetwear brand experience focused on identity, motion, and performance.  
The goal of this project is to create a fast, immersive, and visually dominant digital experience for a fictional fashion brand.

OBSCURA; kimlik, motion (hareket) ve performans odağında tasarlanmış premium bir streetwear marka deneyimidir.  
Bu projenin amacı, kurgusal bir moda markası için hızlı, etkileyici ve güçlü bir dijital deneyim oluşturmaktır.

### This project explores / Bu proje şunları hedefler

- Brutalist UI/UX approach / Brutalist UI/UX yaklaşımı  
- Premium motion design / Premium motion tasarımı  
- Editorial layouts / Editorial tasarım düzeni  
- High-performance frontend experience / Yüksek performanslı frontend deneyimi  
- Responsive design / Responsive tasarım

---

## Tech Stack / Kullanılan Teknolojiler

- React / Next.js  
- Tailwind CSS  
- Framer Motion

---

## Goal / Projenin Amacı

To build a visually dominant, performance-focused streetwear experience that feels premium, modern, and memorable.

Premium, modern ve akılda kalıcı hissettiren; performans odaklı ve güçlü bir streetwear deneyimi oluşturmak.
