# OBSCURA

A brutalist streetwear website concept built by NexBoom Studios.
NexBoom Studios tarafından geliştirilen brutalist streetwear website konsepti.

---

## Overview / Proje Hakkında

**OBSCURA** is a premium, high-energy streetwear web experience focused on identity, motion, and performance.

Designed to feel aggressive, modern, and immersive, the project explores how strong visual identity, brutalist design, and smooth interactions can create a memorable digital brand experience.

**OBSCURA**, kimlik, hareket (motion) ve performans odağında tasarlanmış premium bir streetwear website deneyimidir.

Agresif, modern ve etkileyici bir his yaratmak amacıyla geliştirilen bu proje; güçlü görsel kimlik, brutalist tasarım yaklaşımı ve akıcı etkileşimlerin dijital marka deneyimini nasıl güçlendirebileceğini keşfeder.

## Built Without / Kullanılmayan Teknolojiler

This project intentionally avoids unnecessary frameworks and dependencies in order to maintain performance, flexibility, and full design control.

Bu proje; performans, esneklik ve tam tasarım kontrolü sağlamak amacıyla gereksiz framework ve bağımlılıklardan bilinçli olarak kaçınır.

**Not used / Kullanılmayanlar**

* Tailwind CSS
* Bootstrap
* React / Vue / Angular
* jQuery
* Sass / Less
* UI libraries / hazır UI kütüphaneleri
* Heavy animation libraries / ağır animasyon kütüphaneleri

---

## Why Vanilla? / Neden Vanilla?

**Full control** — Every interaction, spacing, animation, and layout is handcrafted.

**Performance first** — Only necessary code is written, avoiding unnecessary overhead.

**Portfolio quality** — Demonstrates NexBoom Studios’ ability to build optimized and custom frontend experiences from scratch.

### Türkçe

**Tam kontrol** — Her spacing, animasyon ve etkileşim sıfırdan oluşturuldu.

**Önce performans** — Yalnızca gerekli kod yazıldı, gereksiz yüklerden kaçınıldı.

**Portfolio kalitesi** — NexBoom Studios’un optimize ve özgün frontend deneyimleri geliştirme yaklaşımını gösterir.
